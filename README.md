# Amazon Config API

AWS Config provides a detailed view of the configuration of AWS resources in your AWS account. This includes how the resources are related to one another and how they were configured in the past so that you can see how the configurations and relationships change over time. AWS Config enables you to assess, audit, and evaluate the configurations of your AWS resources for compliance and security governance.

## APIs

- **Amazon Config API** - The AWS Config API provides programmatic access to manage configuration recording, evaluate resource compliance against rules, query resource configurations, and track configuration changes across your AWS infrastructure.
